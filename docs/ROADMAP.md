> 本路线图会随进度更新（勾选已合并的 PR）。

## PR 规范（自我约束）

- 标题：一句话说明本 PR 新增/修改了什么。
- 功能描述：作用与使用方式。
- 实现思路：技术选型或核心逻辑。
- 测试方式：如何验证该功能正常运行。
- 合并后 `main` 保持可运行。
